# Knipper

A wrapper around the C-library for the blink(1) USB Status LED by ThingM. 

## Installation

Add this line to your application's Gemfile:

    gem 'knipper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install knipper
